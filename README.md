Weather App
Overview:
A modern, responsive weather application that provides real-time weather information and forecasts. The app features a clean, glassmorphism-style interface with a beautiful background design.

Key Features:

City Search - Users can search for any city worldwide to view its weather conditions

Geolocation Support - Get weather data for the user's current location with a single click

Current Weather Display - Shows:

Temperature in Celsius
"Feels like" temperature
Weather condition description
Humidity percentage
Wind speed (km/h)
Atmospheric pressure (mb)
Visibility (km)
7-Day Forecast - Displays daily forecasts with maximum and minimum temperatures for the next 7 days

Real-time Data - Fetches weather data from the Open-Meteo API, a free and open-source weather API

Multi-language Support - Weather descriptions in English with proper weather code interpretation

Responsive Design - Works seamlessly on desktop, tablet, and mobile devices

Beautiful UI - Modern glassmorphism design with smooth animations and transitions

Technology Stack:

HTML5 for structure
CSS3 for styling with gradient effects and animations
Vanilla JavaScript for functionality
Open-Meteo API for weather data and geocoding
